# CV Personnel Interactif — Portfolio Web

Projet réalisé dans le cadre du module Programmation Web 2 (Licence L2).  
Ce projet consiste à créer un CV personnel interactif accessible en ligne via GitHub Pages.

---

## Démo en ligne

Lien du site :
https://selkarni9903-source.github.io/

---

## Objectif du projet

L’objectif de ce projet est de concevoir un CV web interactif en utilisant les technologies suivantes :

- HTML5 et CSS3 pour la structure et la mise en forme
- JavaScript et jQuery pour l’interactivité et les animations
- ReactJS pour la création de composants dynamiques réutilisables
- GitHub Pages pour l’hébergement du site

Ce projet permet de mettre en pratique les notions vues en cours et de réaliser une interface web moderne et fonctionnelle.

---

## Technologies utilisées

### Front-end
- HTML5 (structure sémantique)
- CSS3 (design, layout, responsive design)
- JavaScript (logique et interactivité)

### Bibliothèques
- jQuery (animations DOM, filtres, validation du formulaire)
- ReactJS (utilisé via CDN avec Babel pour des composants fonctionnels simples)

### Déploiement
- GitHub Pages

---

## Fonctionnalités

### Interface utilisateur
- Design moderne et professionnel
- Mode clair et mode sombre
- Site responsive compatible mobile, tablette et desktop
- Animations CSS fluides

### Interactivité avec JavaScript et jQuery
- Effet de texte dynamique (typing effect)
- Animation des sections lors du scroll
- Barres de compétences animées
- Filtrage dynamique des projets
- Effets hover sur les éléments
- Validation du formulaire de contact

### Composants ReactJS
- Composant SkillBar pour les compétences
- Composant ProjectCard pour les projets
- Composant ContactForm pour le formulaire de contact
- Utilisation de props
- Rendu dynamique avec map

## Sections du CV

### En-tête
- Nom complet
- Photo de profil
- Titre de formation
- Email et liens GitHub

### À propos
- Présentation personnelle
- Parcours académique
- Objectifs professionnels

### Compétences
- Liste des compétences techniques et personnelles
- Représentation visuelle avec barres de progression

### Formation
- Parcours académique sous forme de timeline

### Projets
- Minimum deux projets réalisés
- Description détaillée
- Technologies utilisées
- Lien vers GitHub ou démo

### Contact
- Formulaire avec nom, email et message
- Validation des champs avec jQuery

## Déploiement sur GitHub Pages

Étapes de déploiement :

1. Création d’un dépôt GitHub nommé username.github.io
2. Ajout des fichiers du projet
3. Push sur la branche main
4. Activation de GitHub Pages dans les paramètres
5. Accès au site via une URL publique
