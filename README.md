# CV Personnel Interactif

Projet réalisé en HTML, CSS, jQuery et ReactJS.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- jQuery
- ReactJS

## Fonctionnalités

- Dark / Light mode
- Animations jQuery
- Typing effect
- Timeline animée
- Barre de progression animée
- Filtre des projets
- Formulaire validé

## Déploiement

Projet hébergé avec GitHub Pages.
